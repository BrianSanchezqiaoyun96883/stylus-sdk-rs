# ERC-721 helper scripts

This folder contains a script to quickly mint an ERC-721 token.configured in .env.

## How to use it

1. Install the dependencies

```shell
yarn
```

2. Copy and fill the .env file

```shell
cp .env.example .env
```

3. Run the minting script

```shell
yarn mint
```
